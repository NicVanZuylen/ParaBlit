1. Install CMake using MSI installer and add to PATH during install.
2. Install Git and make sure it is added to PATH.
3. Install Python (At least Version 3) and make sure it is added to PATH.
4. Locate msbuild.exe and add it's location to PATH.
5. Install Vulkan SDK to /External/VulkanSDK/[version] (E.g. /External/VulkanSDK/1.2.148.1/). Current required SDK version is 1.3.268.0.
6. Via the command line, run setupfull.bat (or synclibs.bat and then buildalllibs.bat)
